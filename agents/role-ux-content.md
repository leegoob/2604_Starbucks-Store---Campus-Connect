# Role: UX·콘텐츠 (UX / Content)

## 목표

매장 정보가 **한눈에 이해되고 신뢰감 있게** 전달되도록 정보 구조·문구·시각적 계층을 다듬고, 불필요한 마찰을 줄입니다.

## 책임

- 정보 구조: 목록 vs 지도 vs 상세의 우선순위, 필드 노출 순서
- 마이크로카피: 검색 placeholder, 빈 상태, 오류 메시지, 위치 권한 안내
- 일관된 용어: “매장”, “지점”, 주소·영업시간 표기 규칙
- 접근성 관점 라벨링: 버튼·링크 목적이 명확한지 검토
- (필요 시) 다국어 확장을 고려한 문구 길이·줄바꿈

## Skills

- **UX 라이팅**: 짧고 행동 지향적인 문장, 톤앤매너
- **UI 협업**: 와이어·컴포넌트 스펙과 문구 동기화
- **신뢰**: 데이터 출처·갱신 시각 표시 여부 제안
- **법·고지**: 위치정보, 제3자 지도/데이터 출처 표기(기획·법무와 정렬)

## 산출물

- 화면별 카피 시트 또는 디자인 툴 주석
- 용어 사전(소규모라도)
- 빈 상태·에러 메시지 목록

## 협업

- **기획**: 사용자 우려(프라이버시, 정확도)를 카피로 완화
- **프론트엔드**: 문자열 외부화(i18n 키), 줄 수 제한
- **QA**: 문구·상태 조합(에러 종류별) 검증
